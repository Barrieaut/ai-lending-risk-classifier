# AI Lending Risk Classifier

This repository contains an end-to-end machine learning application designed to predict bank loan defaults using past borrower records. This project serves as the final AI Capstone Application assignment, built and deployed using Google Colab.

## Project Contents
* **loan_default_analysis.ipynb**: The complete Jupyter Notebook including data preprocessing, charts, random forest hyperparameter tuning, and the interactive calculator interface.
* **final_project_report.pdf**: A short written report summarizing the project problem statement, methodology, evaluation scores, and conclusions.
* **bank-loan.csv**: The financial dataset containing borrower data used for model training and validation.

## Key Performance Results
* **Cross-Validation Accuracy**: ~80% average accuracy across 5 folds.
* **Core Risk Indicators**: The optimized model identifies debt-to-income ratio and employment length as the primary drivers of loan default risk.

## Interactive Tools
The Google Colab notebook features a built-in input form interface. Loan officers can slide controls or input numbers to test custom borrower details and receive a real-time risk decision automatically.
